# BioNet
A simple javafx-based tool to visualise protein networks.

## Build
Use [Maven](https://maven.apache.org/download.cgi) to build from source.
```console
sudo apt-get install maven
mvn clean javafx:run
```

## Usage
```console
java -jar BioNet.jar 
```
